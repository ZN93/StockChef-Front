# 🧑‍🍳 StockChef-front

Interface utilisateur de l’application StockChef développée en **React 18** avec **Tailwind CSS**.

## 🚀 Démarrage local

```bash
npm install
npm run dev
```

## 📁 Structure

- `src/` : composants React
- `public/` : fichiers statiques
- `tailwind.config.js` : config Tailwind

## 🧪 Tests

```bash
npm run test
```
